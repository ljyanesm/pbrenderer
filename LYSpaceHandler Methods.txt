List of methods for LYSpaceHandler

- getAABB(vec3 min, vec3 max)		Get objects in AABB
- getPoint(vec3 p)					Get objects in subdivision (either Hash or Octree)
- getSphere(vec3 p, float r)		Get objects in Sphere

- insertObject(LYMesh o)			Insert new objects
- insertPoint(vec3 p, int cat)		Insert new points into subdivision, with some "category"


- deleteObject(LYMesh *o)			Remove object from subdivision
- deletePoint(int p)				Remove point from subdivision (passing the point location) no update
- deletePointUpdate(int p)			Remove point from subdivision (passing the point location) and update
- deleteCategory(int cat)			Remove whole "category"

- update()							Update the subdivision to the new object's configuration
